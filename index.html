🚀 Cómo configurarlo
1
Crear el repo en GitHub
Creá un repo público llamado flight-monitor y subí todos los archivos del ZIP.
2
Crear bot de Telegram
Hablale a @BotFather → /newbot → copiá el token.
3
Obtener tu Chat ID
Hablale a @userinfobot en Telegram → te responde tu ID. Pegalo en Plugins → Telegram.
4
Cargar secreto en GitHub
Settings → Secrets → Actions → New: nombre TELEGRAM_TOKEN, valor = el token.
5
Activar GitHub Pages
Settings → Pages → Branch main, carpeta /dashboard. Dashboard en vivo en ~1 min.
6
Primer run manual
Actions → Flight Monitor → Run workflow. Después corre solo cada 6 hs.
💡 100% gratuito: GitHub Actions da 2000 min/mes gratis. Corriendo cada 6 hs usás ~12 min/mes.
